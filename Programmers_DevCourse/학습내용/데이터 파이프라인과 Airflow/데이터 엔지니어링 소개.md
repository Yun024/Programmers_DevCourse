# 데이터 엔지니어링 소개
## 데이터 조직이 만들어내는 부가가치 1: 데이터 기반 의사결정
#### 부가가치를 만들어내려면 기반이 필요: 데이터 인프라
- 데이터 기반 의사 결정과 제품 개선이 가능하려면?
    - 정제된 데이터가 한 곳에 수집되어 있어야함: 데이터 인프라
- 데이터 인프라
    + 데이터 웨어하우스(혹은 데이터 레이크)
        - 정제된 데이터들이 수집되어 있는 중앙 데이터베이스
        - 구조화된 데이터(혹은 비구조화된 데이터)
    + 데이터 파이프라인(ETL: Extract, Transform, Load)
        - 데이터를 수집해주는 프로세스
        - 데이터 소스 -> 추출 -> 정제 -> 로드 -> 데이터 웨어하우스
